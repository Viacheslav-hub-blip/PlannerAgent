# Внешние MCP tools

Используй этот skill только когда задача требует внешних MCP tools.

## Правила

- MCP tools подключаются при сборке агента встроенными средствами.
- Если MCP-сервис недоступен при старте, агент работает без этих tools.
- Ориентируйся на description конкретного MCP tool и не выдумывай формат ответа.
- Если tool возвращает JSON-строку, кратко объясни пользователю статус, найденные
  источники и ограничения.
