# Термины антифрод-аналитики

| Термин | Значение |
|---|---|
| Сработка | Операция, по которой сработал антифрод. |
| Hit table | Таблица, содержащая только антифрод-сработки. |
| Raw event table | Таблица полной истории клиентских событий. |
| Карточный канал | Операции по картам, POS, E-commerce, ATM, авторизации, токенизации. |
| UKO / не карточный канал | ДБО, ВСП, СБОЛ, СБП, переводы, сервисные операции, операции в отделении. |
| `policy_action` | Первичное решение антифрод-системы. |
| Resolution | Результат подтверждения или расследования. |
| `is_save` | Признак предотвращенного ущерба. |
| CSI | Индекс клиентской удовлетворенности. |
| Негативный CSI | CSI-ответ со значением `score` от `1` до `3`. |
| `epk_id` | Клиентский идентификатор для объединения источников. |
