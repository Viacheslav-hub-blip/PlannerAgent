# Источник CSI

## `source_1.csv`

**Роль:** источник клиентских ответов и оценок CSI.

**Абсолютный путь:** не указан в исходном описании. Используй путь из доступного окружения выполнения.

**Бизнес-смысл:** содержит ответы клиентов на вопросы CSI. Используется для расчета CSI, анализа клиентского негатива и связи с антифрод-событиями.

# Поля

| Поле | Тип | Значение |
|---|---|---|
| `epk_id` | `string` | Идентификатор клиента. Используется для соединения с другими источниками. |
| `question_number` | `int` | Номер вопроса CSI. |
| `answer_text` | `string` | Текст ответа клиента. |
| `score` | `int, 1..5` | Оценка клиента. |

# Правила интерпретации

- `score` от `1` до `3` считается негативным CSI.
- `score` от `4` до `5` можно считать нейтральным или положительным CSI, если в задаче не задана другая логика.
- `answer_text` можно использовать для качественного анализа причины недовольства клиента.
- Источник полезен для связи антифрод-сработок с клиентским опытом.

# Типовые вопросы

Используй источник, чтобы ответить:

- был ли у клиента негативный CSI;
- какие ответы оставлял клиент;
- связаны ли сработки или правила с клиентским негативом;
- какие темы чаще встречаются в негативных ответах.
